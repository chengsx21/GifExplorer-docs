# 需求分析

## 概述

用户端：

- 提供 Gif 主页展示功能
    - 可以浏览个性推荐、站点精选的最新 Gif
- 提供 Gif 检索功能
    - 检索结果默认按相关度排序
    - 可以添加标题、作者、分类、标签等过滤条件
- 点击相应 Gif 后直接跳转至对应的 Gif 详情页
- 支持用户管理个人信息
    - 密码，头像，个性签名
- 用户可以查看浏览历史
- 用户可以将希望查看的新闻添加至稍后再看列表
- 用户可以收藏新闻
    - 系统可以为用户收藏的新闻进行智能摘要
